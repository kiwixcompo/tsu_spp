TSU STAFF PROFILE - QUICK DEPLOY
========================================

FIRST TIME SETUP:
-----------------
Double-click: setup-git.bat
(Only needed once to initialize Git)

TO DEPLOY TO GITHUB & CPANEL:
------------------------------

Option 1 (Recommended):
Double-click: deploy-all.bat
(Does everything automatically)

Option 2 (If Git already set up):
Double-click: deploy.bat

What Happens:
-------------
1. Initializes Git (if needed)
2. Commits all changes to Git
3. Pushes to GitHub repository
4. cPanel automatically deploys to server
5. Sets correct file permissions
6. Your site is live!
